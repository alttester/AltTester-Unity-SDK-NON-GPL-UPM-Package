# AltTester Unity SDK Non-GPL (UPM Package)

This repository hosts the Non-GPL version of the AltTester Unity SDK. It is formatted specifically as a Unity Package Manager (UPM) package, allowing you to easily import, update, and manage the SDK directly within your Unity project.

## Prerequisites

* **Git:** Your machine must have Git installed and added to your system's PATH environment variable. Unity requires Git to be installed in order to fetch and download packages via Git URLs.

## Installation via Unity Package Manager

To import the AltTester Unity SDK Non-GPL package into your project, follow these steps:

1. Open your project in the **Unity Editor**.
2. Open the Package Manager by navigating to **Window** > **Package Manager** in the top menu.
3. Click the **`+`** button located in the top-left corner of the Package Manager window.
4. Select **Add package from git URL...** from the dropdown menu.
5. Paste the following URL into the text field:
   ```text
   https://github.com/alttester/AltTester-Unity-SDK-NON-GPL-UPM-Package.git
   ```
6. Click Install.

Unity will now fetch and install the package. Once the process is complete, you will see the AltTester Unity SDK listed in your Package Manager under your project packages.
